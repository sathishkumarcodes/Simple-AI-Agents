# Simple-AI-Agents

A collection of lightweight, production-ready AI Agents built using n8n, Google Gemini, and simple automation workflows.
Each agent is designed to solve a single real-world problem with minimal configuration and maximum usability.

## Included AI Agents

- [Task & Grocery List Manager](https://github.com/sathishkumarcodes/Simple-AI-Agents/tree/103be6cbc690e908a76fdb34486bc45522614bca/Task%20%26%20Grocery%20List%20Manager) - 
A WhatsApp-powered AI assistant that manages your grocery list and task list using natural language. Users can add, remove, and view items, and everything stays synced in Google Sheets. Built with: WhatsApp Trigger → n8n AI Agent → Gemini → Google Sheets.

- [Subscription Manager](https://github.com/sathishkumarcodes/Simple-AI-Agents/tree/533d7b97533b5fbd1af56cd90c880c4047f38ca5/Subscription%20Manager) -
Automated subscription tracker using n8n workflows to detect recurring costs, forecast next payments, send weekly reports, and trigger urgent alerts for annual subscriptions that are recurring or nearing expiration.
